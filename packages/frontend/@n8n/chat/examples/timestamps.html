Custom Timestamp Styling

You can customize the appearance of timestamps using CSS variables:

/* Custom timestamp styling */ :root { --chat--timestamp--font-size: 0.7rem; --chat--timestamp--color: #999; --chat--timestamp--opacity: 0.8; } /* Different styles for user vs bot timestamps */ .chat-message-from-user .chat-message-timestamp { text-align: right; color: rgba(255, 255, 255, 0.8); font-style: italic; } .chat-message-from-bot .chat-message-timestamp { text-align: left; color: #666; font-style: italic; }
