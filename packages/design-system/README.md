# n8n-design-system

A component system for [n8n](https://n8n.io) using Storybook to preview.

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
pnpm storybook
```

### Build static pages

```
pnpm build:storybook
```

### Run your unit tests

```
pnpm test:unit
```

### Lints and fixes files

```
pnpm lint
```

### Build css files

```
pnpm build:theme
```

### Monitor theme files and build any changes

```
pnpm watch:theme
```
