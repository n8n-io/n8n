# n8n-design-system

A component system for [n8n](https://n8n.io) using Storybook to preview.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run storybook
```

### Build static pages

```
npm run build:storybook
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```

### Build css files

```
npm run build:theme
```

### Monitor theme files and build any changes

```
npm run watch:theme
```
