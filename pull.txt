## Summary

This PR adds a new option "Automatically Passthrough Binary Audios" to the AI Agent node. This allows audio files to be automatically passed through to multimodal models (like OpenAI and Gemini), mirroring the existing functionality for images.

### Key Changes:
- Added `passthroughBinaryAudios` to the AI Agent node properties in `options.ts`.
- Updated `extractBinaryMessages` in `common.ts` to detect audio files and format them using the `input_audio` structure compatible with major AI providers.
- Updated Agent V1, V2, and V3 execution logic to respect the new toggle.
- Added comprehensive unit tests in `commons.test.ts` and `prepareItemContext.test.ts`.

### How to test:
1. Connect a multimodal model (e.g., OpenAI GPT-4o) to an AI Agent node.
2. Provide audio files (e.g., mp3, wav) as input to the node.
3. Enable "Automatically Passthrough Binary Audios" in the node options.
4. Run the node and verify that the audio is sent to the model and correctly processed.
5. Verify that disabling the option prevents the audio from being sent.
