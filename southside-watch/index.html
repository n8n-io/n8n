SCC Approves $29.4M Transmission Upgrade — Cost Passed to All Ratepayers

The Virginia State Corporation Commission approved a Dominion Energy transmission upgrade in Southampton County. Our Approach

How This Project Works

📋

Public Records First

Every factual claim is sourced and confidence-rated. We cite public records, FOIA responses, meeting minutes, permit filings, deed records, and verified community reports. We clearly separate confirmed facts from community tips under investigation.

⚖️

Pro-Transparency, Not Anti-Tech

This project is about accountability, not opposition to development. Data centers can bring real benefits to economically distressed communities — but only when those benefits are honestly weighed against real costs, with full public disclosure before any vote.

🏘️

Resident-First

Southside Virginia communities deserve the same transparency, the same protections, and the same seat at the table as any other region. We exist because those communities have historically been the last to learn what's happening and the last to be asked.

About the Data on This Site

All project data on this site is compiled from public records, FOIA requests, deed research, corporate registration records, and community reports. Where information is unconfirmed, it is labeled with confidence ratings (Low, Medium, Verified) and placeholders where records are pending (FOIA Pending, Source Needed). This site is a living document — it will grow as more public records are collected. If you have information that can improve the accuracy of any entry, please submit it here.
