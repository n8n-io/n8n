### Common Patterns

#### Permission Checking
```typescript
// Controller level
@RestController()
@Authorized()
class WorkflowController {
  @Get('/:id')
  @RequirePermission('workflow', 'read')
  async getWorkflow() {}
}

// Service level
if (!user.hasPermission('workflow', 'update', workflow)) {
  throw new ForbiddenError();
}
```
