🔴 → 🟢 TDD discipline

Every implementation task in this kanban follows the same loop:

  1. Write the failing test first. The test file path is named in each card.
  2. Run it. Confirm it fails for the right reason ("Cannot find module" means you haven't even imported correctly — write more test first).
  3. Implement minimal code to make the test pass. Don't over-build.
  4. Run again. Confirm green.
  5. Refactor if needed — keep tests green throughout.

Backend tests: Jest (pnpm --filter n8n test path/to/file.test.ts). CLI tests: Jest with nock for HTTP mocks (pnpm --filter @n8n/cli test). SDK tests: Jest with nock (pnpm --filter @n8n/sdk test). Frontend: vitest.

🔍 Peer-review findings — applied to plan

Codex CLI was unavailable; validation ran via senior-engineer agent reading the actual codebase. Six material findings, all applied.

1Phase 1.1: do NOT copy test-workflow.tool.ts verbatim
That file seeds from a trigger node and throws when none exists. Use WorkflowRunner.run with pre-seeded nodeExecutionStack[0]. Recipe inline in 1.1.
2Phase 0.3: drop RemoteBaseCommand
@Command() decorator unconditionally calls Service()(target). Solution: plain @Command classes that don't override init.
3Phase 4.3 codegen: cut on day 1
Hand-roll .d.ts for the 8 demo nodes — that IS the path.
4Phase 1.4 consent UI doesn't render scopes today
6h frontend+backend, not "strings." Skip for hackathon. Deferred to production.
5@GlobalScope decorator required
Every authenticated route gates with @ProjectScope/@GlobalScope. May need to add node:execute to @n8n/permissions.
6Scope enforcement in MCP middleware
Adding to SUPPORTED_SCOPES ≠ enforced. Deferred to production must-fix.

Risks ranked

1additionalData wiring in 1.1 doesn't decrypt credentials
Most likely blocker. Mitigation: use the recipe in 1.1 — pre-seeded nodeExecutionStack, route via processRunExecutionData.
2Tool-search ranking is mediocre
Keyword match on short queries is noisy. Mitigation: rank-boost connected credentials; limit to top 8.
3OAuth loopback fails behind VPN
Mitigation: PAT fallback for CLI + SDK.
4Claude Desktop transport hiccups during the demo
Mitigation: Loom backup pre-recorded; curl fallback in a second terminal.
5Mode value breaks an unexpected consumer
Mitigation: full pnpm typecheck immediately after Phase 0; alias to 'manual' everywhere except telemetry.
6Slack channel resourceLocator
Hero demo fails with channel name. Pre-flight prints real IDs.
