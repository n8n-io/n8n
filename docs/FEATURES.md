# 启用的企业功能清单

本文档列出所有默认启用的企业版功能及其配置状态。

## 🔐 认证与安全

### LDAP 集成
- **状态**: ✅ 完全启用
- **配置**: `enterprise.ldap = true`
- **功能**:
  - 企业目录服务集成
  - 自动用户同步
  - 组映射和权限管理
  - 单点登录支持

### SAML 单点登录
- **状态**: ✅ 完全启用
- **配置**: `enterprise.saml = true`
- **功能**:
  - SAML 2.0 协议支持
  - 与企业 IdP 集成
  - 自动用户配置
  - 自定义属性映射

### OIDC 单点登录
- **状态**: ✅ 完全启用
- **配置**: `enterprise.oidc = true`
- **功能**:
  - OpenID Connect 认证
  - OAuth 2.0 授权流程
  - 与主流 IdP 兼容
  - 令牌刷新机制

### 多因素认证 (MFA)
- **状态**: ✅ 完全启用
- **配置**: `enterprise.mfaEnforcement = true`
- **功能**:
  - TOTP 时间基础令牌
  - 强制 MFA 策略
  - 备用恢复代码
  - 用户级 MFA 管理

### 高级权限管理
- **状态**: ✅ 完全启用
- **配置**: `enterprise.advancedPermissions = true`
- **功能**:
  - 细粒度资源权限
  - 自定义角色定义
  - 资源级访问控制
  - 继承权限模型

## ⚠️ 注意事项

1. **AI 助手**: 使用空证书和未知消费者 ID，部分云端功能可能需要额外配置
2. **云服务**: 已移除版本通知和动态横幅等云服务依赖
3. **安全性**: 需自行实现额外的访问控制和安全措施
4. **配置**: 所有功能都可通过环境变量或配置文件进一步定制
