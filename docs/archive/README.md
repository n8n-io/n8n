# 归档文档

本目录包含详细的历史文档，用于深入了解项目的完整开发历程。

## 文档说明

- **[FULL-CHANGELOG.md](./FULL-CHANGELOG.md)** (1270 行)
  - 完整的开发历史记录
  - 包含所有版本的详细变更
  - 按时间倒序排列

- **[DETAILED-CHANGES.md](./DETAILED-CHANGES.md)** (331 行)
  - 详细的代码变更说明
  - 删除的核心文件列表
  - 各阶段的修改清单

- **[DETAILED-TECHNICAL.md](./DETAILED-TECHNICAL.md)** (815 行)
  - 深入的技术实现细节
  - 完整的代码示例
  - 策略演变历史

## 使用建议

这些文档主要用于：
1. **深入研究** - 了解具体实现细节
2. **历史追溯** - 查找过去某个时间点的变更
3. **学习参考** - 研究技术决策和演进过程

日常使用建议查看父目录的精简文档：
- [最近变更](../RECENT-CHANGES.md) - 最近几个月的重要更新
- [技术概述](../TECHNICAL-OVERVIEW.md) - 核心技术概述
