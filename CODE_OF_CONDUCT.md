# 贡献者契约行为准则

## 我们的承诺

为了培养一个开放和欢迎的环境，作为贡献者和维护者，我们承诺让参与我们项目和社区的每个人都能免受骚扰，无论年龄、体型、残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性认同和性取向。

## 我们的标准

有助于创造积极环境的行为示例包括：

- 使用欢迎和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 专注于对社区最有利的事情
- 对其他社区成员表现出同理心

参与者不可接受的行为示例包括：

- 使用性化的语言或图像以及不受欢迎的性关注或
  追求
- 恶意攻击、侮辱/贬损评论以及个人或政治攻击
- 公开或私人骚扰
- 未经明确许可，发布他人的私人信息，如物理或电子
  地址
- 在专业环境中可能被认为不合适的其他行为

## 我们的责任

项目维护者有责任澄清可接受行为的准则，并期望在
对任何不可接受的行为实例采取适当和公平的纠正措施。

项目维护者有权利和责任删除、编辑或
拒绝评论、提交、代码、wiki 编辑、问题和其他贡献
不符合本行为准则，或暂时或
永久禁止任何贡献者参与他们认为不合适、
威胁性、攻击性或有害的行为。

## 适用范围

本行为准则适用于项目空间内和公共空间中
当个人代表项目或其社区时。代表的示例
项目或社区包括使用官方项目电子邮件
地址、通过官方社交媒体账户发布，或作为指定
在线或离线活动的代表。项目维护者可以
进一步定义和澄清项目的代表。

## 执行

可以通过联系项目团队 jan@n8n.io 来报告
滥用、骚扰或其他不可接受的行为实例。所有
投诉将被审查和调查，并将得到被视为必要且适合情况的回应。项目团队有义务对事件报告者的机密性
保密。具体执行政策的更多细节可以单独发布。

不真诚地遵循或执行行为准则的项目维护者
可能会面临由项目领导层其他成员决定的
临时或永久后果。
